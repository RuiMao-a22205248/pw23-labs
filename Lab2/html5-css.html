HTML5

O HTML5 é a nova versão do HTML4,addicionou alguns Elementos Semânticos HTML5

Elemento semântico Uso ou Não Descrição
header 😀 Parte cima de página
nav 😀 Parte Menu
main 😀 Parte principal da página
article ❌ Não uso
section 😀 Dentro parte main
aside ❌ Não uso
footer 😀 Parte baixo da página

CSS

CSS é a linguagem que usamos para estilizar uma página Web

Seletor Uso ou Não Descrição
universal 😀 Fonte para todas página
tipo 😀 Parte Menu
classe 😀 Parte principal da página
id 😀 Parte menu de página
atributo 😀 Parte Menu de página
pseudo-classe 😀 Parte Menu de página
pseudo-elemento 😀 Parte h1 da página


Combinador de seletor Uso ou Não Descrição
Descendentes: espaço ❌ Não uso
Filhos: > ❌ Não uso
Adjacentes: ~ ❌ Não uso
Imediatamente adjacente: + ❌ Não uso
Agrupar: , ❌ Não uso
